# AI Plugins

This repo contains AI agent tools: MCP servers, skills, packages, and plugins.

## Structure

- `mcp-servers/` — MCP protocol servers (Python/TypeScript)
- `skills/` — Agent skills (Markdown + code)
- `packages/` — Pi packages
- `plugins/` — Claude Code plugins

## Languages

- Python (uv)
- TypeScript (pnpm)

## Commit style

- `feat:` new tools or features
- `fix:` bug fixes
- No Co-Authored-By lines in commits
