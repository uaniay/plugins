# Packages

Pi packages — each subdirectory is an independent package distributable via npm or git.

## Structure

```
packages/
├── <package-name>/
│   ├── package.json    # With "pi" manifest
│   ├── skills/         # Optional
│   ├── extensions/     # Optional
│   ├── prompts/        # Optional
│   └── README.md
```
